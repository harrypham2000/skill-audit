From `skill-audit/`: `npm install` to install deps, `npm run dev` to run the CLI from source with tsx, `npm run build` to compile TypeScript to `dist/`, `npm run start` to run the compiled CLI. Audit usage from package directory: `npx skills-audit`, `npx skills-audit -v`, `npx skills-audit --json`, `npx skills-audit --threshold 5.0`, `npx skills-audit --no-deps`, `npx skills-audit --project`. There are currently no lint or test scripts defined.
