`skill-audit` is a TypeScript/Node.js CLI that audits installed AI agent skills for security risks. It discovers skills via `npx skills list --json`, parses `SKILL.md` with gray-matter, scans bundled files with heuristic regex rules, optionally runs Trivy on the skill directory, computes a risk score, and reports grouped results by agent. Top-level repo also contains a design/planning document for a broader AI Agent Skill Security Auditing solution.
